Caesar Cipher (Կեսարի ծածկագիր) 

[Armenian] Սա Python լեզվով գրված պարզ հրամանային տողի (CLI) ծրագիր է,
որն իրականացնում է հանրահայտ Կեսարի ծածկագիրը (Caesar Cipher): 
Ծրագիրը թույլ է տալիս գաղտնագրել (encode) և վերծանել (decode) տեքստային
հաղորդագրությունները՝ տառերը տեղաշարժելով օգտատիրոջ կողմից նշված քայլերի քանակով:

[English] This is a simple Command Line Interface (CLI) application written in Python 
that implements the famous Caesar Cipher algorithm. It allows users to encrypt
(encode) and decrypt (decode) text messages by shifting letters by a specified number.
